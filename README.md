# AppWeb
# 🌐 Web Applications (React + Vite + Cursor)

Repositorio de aplicaciones web desarrolladas durante mi formación en Ingeniería en Sistemas Computacionales, enfocadas en la creación de interfaces modernas a partir de diseños en Figma e implementación con React.

## 🚀 Descripción

Este repositorio contiene múltiples interfaces web desarrolladas como parte de mi formación académica, donde se aplican conceptos de desarrollo frontend, diseño UI/UX y herramientas modernas de desarrollo asistido por inteligencia artificial como Cursor.

Los proyectos están basados en diseños de Figma y fueron transformados en aplicaciones funcionales utilizando tecnologías actuales del ecosistema web.

## 🛠 Tecnologías utilizadas

- React
- Vite
- JavaScript
- HTML5
- CSS3
- Cursor (programación asistida por IA)
- Figma (diseño de interfaces)

## 📂 Proyectos incluidos

### 🔹 Interfaz Web UJAPath
- Desarrollo basado en diseño de Figma
- Implementación de estructura modular con React
- Enfoque en diseño limpio y organizado

### 🔹 Veterinary Clinic Interface
- Interfaz para sistema de clínica veterinaria
- Organización de componentes y vistas
- Adaptación de diseño UI a código funcional

### 🔹 LogicPlay Web UI Design
- Diseño centrado en experiencia de usuario
- Aplicación de estilos modernos
- Construcción de interfaz interactiva

## 🧠 Enfoque de desarrollo

- Transformación de diseños UI (Figma) a código funcional
- Uso de herramientas de IA (Cursor) para optimización del desarrollo
- Aplicación de buenas prácticas en frontend
- Desarrollo de interfaces responsivas y estructuradas

## 📦 Estructura del repositorio

Los proyectos se encuentran comprimidos en archivos `.zip`.  
Cada archivo contiene una aplicación independiente.
